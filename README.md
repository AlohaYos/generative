# Generative

## Rrequirements

- Node.js: version 14 or later.
- yarn: version 1.22.19

## WebUI

Installation

```
git clone git@github.com:Cryptocoders-wtf/generative.git
cd generative
yarn install
```
Running
```
yarn run serve
open http://localhost:8080
```

## Solidity backend

Installation

```
cd contract
npm install
```

Deployment
```
npx hardhat run ./scripts/deploy_splatter.ts
```
